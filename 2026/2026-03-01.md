- **来源**: [arXiv:2602.21224](https://arxiv.org/html/2602.21224v1)
- **作者**: Yuetao Chen 等（香港中文大学、滑铁卢大学等）
- **摘要**: 提出在 hidden state 层面进行自回归预测，将 token 信息注入推迟到 hidden state 生成之后，实现 draft hidden states 的重用。

### MARS: Margin-Aware Speculative Verification
- **来源**: [arXiv:2601.15498](https://arxiv.org/html/2601.15498v1)
- **摘要**: 提出 Logit Ratio 作为置信度指标，当 target model 对 top candidates 区分度小时自适应接受 draft token。
- **结果**: 在 8B 到 235B 模型上测试，最高 **4.76× 加速**
