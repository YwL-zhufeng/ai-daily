# AI日报 - 2026年3月5日

## 大模型更新

**OpenAI 发布 GPT-5.3 Instant 模型**  
OpenAI于3月3日发布GPT-5.3 Instant，对即时响应模型进行全面优化。新模型显著减少不必要的拒绝回答和免责声明，优化网络信息筛选与综合能力，对话风格更自然直接。同时推出GPT-5.3-Codex编程模型家族。GPT-5.2 Instant将于6月3日停用。  
来源：[新浪财经](https://finance.sina.cn/stock/jdts/2026-03-04/detail-inhpuskr0270752.d.html)

**DeepSeek V4 即将发布**  
据金融时报报道，DeepSeek将于3月首周发布新一代多模态大模型V4。该模型采用原生多模态架构，支持图片、视频和文本生成，拥有100万token超长上下文窗口。V4将优先适配华为昇腾、寒武纪、海光等国产AI芯片，算力利用率提升至85%，部署成本降至英伟达方案的1/3。  
来源：[雪球](https://xueqiu.com/1438272693/377602771)、[东方财富](https://finance.eastmoney.com/a/202603023658447220.html)

**Anthropic Claude Sonnet 4.6 发布**  
Anthropic于2月18日发布Claude Sonnet 4.6，被称为"史上最强Sonnet"。该模型在多项基准测试中接近旗舰Opus 4.6水平，SWE-bench测试达79.6%，OSWorld计算机操作测试得72.5分。定价仅为Opus的五分之一（$3/$15每百万token），支持100万token上下文窗口（测试版）。  
来源：[DoNews](https://www.donews.com/news/detail/4/6432436.html)

**Claude Opus 4.6 推出 Agent Teams 功能**  
Claude Opus 4.6于2月5日发布，新增Agent Teams功能，允许多个Claude Code实例协同工作。该模型在SWE-bench Verified上表现优异，支持自适应思考模式，可根据任务复杂度自动调整推理深度。Anthropic CEO确认Claude 5将于2026年Q2发布。  
来源：[Taskade](https://www.taskade.com/blog/anthropic-claude-history)

**Google Gemini 3.1 Pro 持续领先**  
Gemini 3.1 Pro在16项主要基准测试中领先13项，ARC-AGI-2逻辑测试得分77.1%，GPQA Diamond专家级科学知识测试达94.3%。定价为$2/$12每百万token，实现前沿性能与大众化价格的结合。  
来源：[Mean CEO Blog](https://blog.mean.ceo/new-ai-model-releases-news-march-2026/)
